ODD catalog service

Privacy policy

ODD uses a public static catalog at odd.regionallyfamous.com/catalog/v1/ so the WordPress plugin can show available wallpapers, icon sets, cursors, widgets, and apps.

When a site administrator opens or refreshes the ODD Shop, runs the starter pack, installs catalog content, or repairs an installed bundle, the WordPress site may request static JSON, image, icon, or bundle files from this domain.

Those HTTPS requests include ordinary web request metadata such as IP address, user agent, requested URL, referrer if supplied by the browser/server, and timestamp. ODD does not send site content, WordPress user account details, cookies, analytics events, license keys, or diagnostic reports to the catalog service.

The site is hosted as static web content. Hosting providers may keep standard server logs for security, abuse prevention, and reliability. Those logs are not used by ODD for tracking individual WordPress users.

Questions can be opened at github.com/RegionallyFamous/odd/issues.
