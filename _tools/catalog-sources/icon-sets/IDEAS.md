# ODD Icon Set Rebuild Ideas

The catalog is intentionally reset to `odd-default-icons` while we rebuild
the first-party icon sets one at a time.

## First Set Candidate: Chunky Signal

- Slug: `chunky-signal`
- Goal: a normal, readable Desktop Mode metaphor set with ODD personality
  coming from bold raster shape language, not runtime effects.
- Shape rule: each icon should read as one large silhouette plus one supporting
  interior cut at most.
- Line rule: thick dark rim, thick inner lines, no hairlines, no tiny dots,
  no dense texture, no labels, no letters, no boxed tiles.
- Scale rule: visible alpha footprint around 84-88% of the canvas, with
  transparent padding reserved for glow/shadow and no edge clipping.
- Palette: off-white fill, deep ink outline, electric cyan shadow, hot pink
  offset accent used sparingly.
- Finish: flat raster sticker material with a slight uneven hand-cut edge and
  soft outside glow.

## Glyph Notes

- odd: animated ODD eye launcher, recolored to the set palette.
- my-wordpress: one chunky desktop dashboard gauge with a fat needle.
- content-graph: connected content nodes with thick graph lines.
- recycle-bin: simple open bin, no small recycling mark.
- fallback: chunky four-point spark or abstract ODD mark.
